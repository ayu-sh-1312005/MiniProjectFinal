Problem Statement

Design and develop a smart parking system that addresses the challenges of urban congestion, inefficient parking space utilization, and the inconvenience faced by drivers in finding available parking spots in crowded areas. The system should leverage modern technologies to provide an efficient, user-friendly, and automated solution for both drivers and parking lot operators.

Solution

We have proposed a software that allows user to get into the parking lot seemlessly while the system detects its number plate and allots it a parking slot. While exiting it detects the number plate and charges the user according to the time of entry amnd exit.
